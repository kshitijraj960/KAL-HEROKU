Adding to group

Use below Steps to Add Merissa Bot your Group.

  1. Go to MerissaBot and send /start
  2. Then you can see a button called Add to group.
  3. Click on that button and select your groups.

To use MerissaBot in your group, Bolt must be an Admin of your Group. do that use below steps.

  1. Go to your group and click the three dots.
  2. Select Manage group option.
  3. Then go to Administrators.
  4. Finally click on Add Administrator and select Bolt Backer.

Using Commands

Here are the main commands that can be used with Merissa

  • /start - Start to Merissa
  • /help - To get full help menu.

For a easy and fully help you can use /help command in Merissa bot's chat.
